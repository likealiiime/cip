Parameters

initialize

initialize: function(options)

Accepts a configuration Hash.  All configurations are optional.

Configuration

cssClassString.  CSS Classes to give the element
cssStylesHash.  CSS Styles to give the element.  Automatically passed to <CIObject.interpretStyles>
handleString.  Handle by which to access the object.  See <CIApplication.handles>
<CIEvent>String.  The constant name of the CIEvent for which to listen.

Example

new CIObject({
    cssClass: 'small anotherClass',
    cssStyles: { display: 'none', 'background-color': 'red' },
    handle: 'myObject',
    Clicked: function(event) { ... }
});

element(parent)

element: function(parent)

Returns the Element implementation for the CIObject.  It operates in three modes, depending on the parameters

  • No parameters - looks for the Element in the DOM with id of <CIObject.id> using $(<CIObject.id>) and returns that Element or null
  • String or Element - looks for the Element in the DOM using $(String/Element) and passes it to CIObject._makeElement() as the parent, returning an Element.  Fires AddedToDom
  • No parameters, no Element found - returns null

set(property, newValue)

set: function(property,
newValue)

First checks to see if the newValue is different.  If it is, fires <CIEvent.PropertyChanged> passing the property name, newValue, and the current (about to be old) value of property, in that order.  Then sets the property to newValue

Parameters

propertyString.  The property to set
newValueAnything.  The new value

Returns

This CIObject

CIObject.interpretStyles(styles)

CIObject.interpretStyles = function(styles)

Process special CIP styles to CSS styles.  Called automatically by CIObject on its cssStyles config option

CIP Styles

CIFirmWidthValue/Percentage.  Sets min-, max- and width to the specified value.
CIFirmHeightValue/Percentage.  Sets min-, max- and height to the specified value.

Returns

Hash

Example

{ CIFirmWidth: 100, CIFirmHeight: '10%' } =>
{
    'min-width': 100, 'max-width': 100, 'width': 100,
    'min-height': '10%', 'max-height': '10%', 'height': '10%'
}

CIRequest

A general request that wraps Mootools.Request.JSON using CIRequestable so it incorporates CIApplication.baseParams.  This is a non-visual component and will throw an error if element() is called on it

Properties

See <CIRequestable>

Events

  • <CIEvent.RequestFailed>
  • <CIEvent.GotData>
  • <CIEvent.PostedData>
